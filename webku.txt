Fitur AI / Data Science (Tanpa AI Berbayar) 
Sentiment Analysis Berita 
Menggunakan: 
Lexicon Based Sentiment Analysis 
dibuat dengan PHP. 
Output: 
Positive : 60% 
Neutral : 25% 
Negative : 15% 
Contoh: 
Mahasiswa cukup membuat kamus kata (dictionary) positif dan negatif. 
Contoh tabel database: 
positive_words 
id word 
1 growth 
2 increase 
3 profit 
4 stable 
5 improve 
negative_words 
id word 
1 war 
2 crisis 
3 inflation 
4 delay 
5 disaster 
Misalnya berita: 
"Inflation increases while exports decrease due to war." 
Maka program akan: 
Positive: increase 
Negative: inflation 
war 
decrease 
Hasil: 
Positive : 1 
Negative : 3 
Sentiment = Negative 
Coding PHP sederhana: 
$positiveScore = 0; 
$negativeScore = 0; 
foreach ($words as $word) { 
if (in_array($word, $positiveWords)) { 
$positiveScore++; 
} 
if (in_array($word, $negativeWords)) { 
$negativeScore++; 
} 
} 
$sentiment = $positiveScore > $negativeScore 
? "Positive" 
: "Negative";
